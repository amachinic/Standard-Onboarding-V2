# Standard-Onboarding-V2

A lightweight, single-page prototype for a credit-card discovery experience modeled on the same welcome → preferences → recommendations → compare → selected-card flow.

## What’s included
- Welcome screen with intro and hero card preview
- Goal selection screen
- Spend and fee preference screen
- Ranked recommendation screen
- Compare screen
- Final selected-card end state

## Run locally
Open the app directly in a browser, or serve the folder locally with a simple static server.

Example:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000/app.html
